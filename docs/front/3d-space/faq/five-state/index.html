1、 3D空间状态操作

Five 引入了 State 的概念，以方便于更方便地理解当前三维空间的状态。你可以通过 five.state 来获取当前状态或调用 five.setState() 对状态进行修改：

// 获取当前的模态
five.state.mode

// 全景点位游走
five.setState({ panoIndex: 0 })

// 切换当前的模态至 "Floorplan" 空间总览模态
five.setState({ Mode: Five.Mode.Floorplan })

更多详细的介绍可参照 状态控制 。
